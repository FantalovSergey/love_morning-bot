# Описание
Бот для отправки по расписанию ежедневных сообщений любимому человеку (и только ему). Присутствует функционал
администрирования. Логирование в сообщениях админу (DEBUG-ERROR) и в лог-файлы (ERROR-CRITICAL). Бот работает на
серверах [Amvera](https://cloud.amvera.ru/).

# Варианты взаимодействия
## Команда /start:
- Моя Арина: приветственное сообщение, кнопка для получения любовного сообщения прямо сейчас
- Я (админ): кнопка для показа всех возможных сообщений с порядковыми номерами
- Другие пользователи: уведомление об эксклюзивности использования бота
## Кнопка удаления сообщений (только для админа):
- Приглашение к вводу порядковых номеров любовных сообщений для удаления
- Кнопка отмены процесса удаления
- Непосредственно после ввода и отправки осуществляется удаление
